# Proyecto Final modulo 3: Aplicación de consola
Este es mi proyecto del modulo 3 de Fundamentos de JavaScript. Es una aplicacion interactiva que corre en la consola del navegador y combina operaciones matematicas y entrega informacion de algunos personajes de Ragnarok Online.

## Estructura del repositorio

* index.js: Codigo fuente principal ordenado por leccion (de la 1 a la 5).
* index.html: Archivo base para enlazar el codigo de JavaScript.
* capturas/: Carpeta con las imagenes del programa funcionando.

## Caracteristicas del codigo

* Validaciones: Controla que el usuario ingrese numeros reales y evita errores como la division por cero.
* Modularizacion: Cada operacion matematica tiene su propia funcion y se coordinan mediante un switch.
* Arreglos y Objetos: Guarda una lista de personajes (3rd Jobs) en un arreglo de objetos y los recorre usando metodos como forEach y map.

## Como probarlo

1. Abre el archivo index.html en tu navegador.
2. Abre la consola de desarrollador presionando F12 (o clic derecho, Inspeccionar y luego vas a la pestana Consola).
3. Sigue las instrucciones de las ventanas emergentes (prompt) para usar la calculadora y ver los resultados en la consola.
